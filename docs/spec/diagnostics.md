## Назначение

Диагностика должна объяснять ошибку и указывать место в исходной программе.

## Формат

Семантическая диагностика содержит сообщение и позиционную информацию.

```cpp
SemanticDiagnostic
```

## Семантические диагностики

Семантический анализатор накапливает несколько ошибок и возвращает их через `TypeCheckResult`.
