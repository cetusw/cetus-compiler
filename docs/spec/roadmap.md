## Frontend

- Добавить функции и вызовы.
- Добавить декларации типов.
- Добавить пустые блоки.
- Уточнить области видимости.

## Семантика

- Ввести `void` или отдельную модель statement-типов.
- Реализовать проверки member access и index access.
- Улучшить диагностики с диапазонами исходного текста.

## VM

- Реализовать генерацию байткода для AST.
- Поддержать условные переходы для `IfASTNode`.
- Поддержать вывод через стандартную библиотеку.

## Тестирование

- Специфицировать `test`.
- Специфицировать `assert`.
- Специфицировать property-based testing.
- Специфицировать mock-конструкции.
- Добавить verification pipeline.

## Модули

- Реализовать `package`.
- Реализовать `import`.
- Определить видимость символов.

## Документация

- Синхронизировать grammar в документации с `docs/cetus_grammar.txt`.
- Добавить формальные примеры диагностик.
- Добавить спецификацию стандартной библиотеки.
