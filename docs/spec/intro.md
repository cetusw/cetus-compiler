## Назначение языка

CETUS — TDD-ориентированный язык программирования. Тестирование является частью семантики языка, а не внешним соглашением библиотек или фреймворков.

Основная модель CETUS:

- программа содержит код и тесты;
- тесты обнаруживаются компилятором;
- выполнение пользовательской программы допускается только после успешной проверки тестов;
- тесты, проверки свойств и mock-конструкции являются частью языка.

## Принципы CETUS

CETUS проектируется вокруг следующих требований:

- тесты являются первичной частью программы;
- непроверенный публичный код должен быть обнаруживаемым;
- выполнение тестов должно быть воспроизводимым;
- синтаксис должен быть пригоден для статического анализа;
- диагностики должны указывать на причину ошибки, а не только на место сбоя.
