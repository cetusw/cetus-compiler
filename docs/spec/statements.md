## Список операторов

Операторы объединяются в `SequenceASTNode`.

```text
~StmtList~ -> ~StmtList~ ~Stmt~ @sequence
~StmtList~ -> ~Stmt~ @pass_expr
```

## Простые операторы

Простой оператор завершается `;`.

```cetus
a := 2;
printf(a);
a + 1;
```

## Присваивание

```cetus
a := 10;
```

Если имя `a` отсутствует в таблице символов, создаётся новый символ. Если имя уже определено, тип правой части должен совпадать с известным типом.

## Printf

```cetus
printf(1);
printf(a > b);
```

`printf` является встроенной конструкцией вывода.

## If

Поддерживаются формы:

```cetus
if a > b {
    printf(1);
}
```

```cetus
if a > b {
    printf(1);
} else {
    printf(0);
}
```

Вложенные `if` разрешены.

## Блок

Блок содержит список операторов:

```cetus
{
    a := 1;
    printf(a);
}
```
